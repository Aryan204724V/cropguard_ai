# 🌱 CropGuard AI

CropGuard AI is a smart mobile application that helps farmers detect crop diseases using AI.

---

## 🚀 Features
- 📷 Capture crop images
- 🤖 AI-based disease detection
- 🌿 Suggest solutions for crops
- 📊 Easy-to-use interface

---

## 🛠️ Tech Stack
- Flutter
- Dart
- AI / ML (Future Integration)
